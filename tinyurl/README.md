# TinyURL

TneyURL is a customized project based on Educative.io's [Designing a URL Shortening service like TinyURL](https://www.educative.io/courses/grokking-the-system-design-interview).


## Installation

This project use gradle.

```bash
./gradlew build
./gradlew bootRun
```

## Getting Started

```bash
./gradlew bootRun
```
